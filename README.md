# Yandex Music RPC  
### **~Discord Rich Presence~ + OBS Embed для Яндекс Музыки**

Yandex Music RPC - приложение, позволяющее отображать текущий трек в ~**Discord Rich Presence**~ и **в виде карточки в OBS**. Проект подходит для стримеров, людей, которые любят музыку и всех, кто хочет эстетично демонстрировать свой текущий плейлист.

## Возможности

### ~Discord Rich Presence~
- ~Отображение **текущего трека** и **артиста**~
- ~**Обложка альбома** в RPC~  
- ~**Прогресс трека** в реальном времени~  
- ~Статус обновляется автоматически~  

### OBS Embed (HTML Widget)
- Красивый и современный дизайн  
- Показывает:
  - *обложку*  
  - *название трека* 
  - *исполнителей*
  - *прогресс бар*    
- Легко встраивается в OBS
